About Elite Quiz

Elite Quiz is a quiz game that tests your knowledge on a variety of topics. You can play the game in four different modes: Quiz Zone, Guess The Word, Guess The Number, Maths Mania. You can also view the leaderboard to see how you compare to other players.

How To Play

Features

  • Elite Quiz has four different Modes
    •
    1. Quiz Zone
    2. Guess The Word
    3. Guess The Number
    4. Maths Mania
  • Leaderboard
    •
    1. The leaderboard shows the top 10 players
    2. The leaderboard is updated after every game
  • Profile
    •
    1. The profile shows the user's username, email, and points
    2. The profile is updated after every game

Modes

  • Quiz Zone
    •
    1. The user is given 4 options for each question
    2. The user is given 1 point for each correct answer
    3. The user is given 0 points for each incorrect answer
  • Guess The Word
    •
    1. The user is given a word with missing letters
    2. The user is given 1 point for correct answer
    3. The user is given 0 points for incorrect answer
  • Guess The Number
    •
    1. The user is given 5 Guesses to guess the number
    2. The user is given 1 point for each correct answer
    3. The user is given 0 points for each incorrect answer
  • Maths Mania
    •
    1. The user is given 4 options for each Maths question
    2. The user is given 1 point for correct guess
    3. The user is given 0 points for incorrect guess
